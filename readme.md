# Niveditha College of Pharmacy (NCP) Website

A responsive website for Niveditha College of Pharmacy, specializing in the D.Pharma program.

## Features
- **Responsive Header**: Three-slot layout for Logo, Navigation, and CTA.
- **Hero Section**: Customized backgrounds with specialized course messaging.
- **Academic Info**: Detailed cards for intake and program specifics.

## Built With
- HTML5
- CSS3 (Flexbox)
