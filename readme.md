# Niveditha College of Pharmacy (NCP) Website

A comprehensive, professional, and responsive website for Niveditha College of Pharmacy, Chintamani. This project showcases the college's legacy, academic rigor, and vibrant campus life.

## 🛠️ Features
- **Multi-Page Architecture**: Five dedicated pages including Home, About, Academics, Admissions, and Contact.
- **Responsive UI**: A custom "Three-Slot" flexbox header and a mobile-friendly grid system for academic cards.
- **Infrastructure Showcases**: Specialized sections for Laboratory facilities and a Resource Center (Library) with "zigzag" layouts.
- **Interactive Contact Page**: Features an embedded live Google Map and a functional Admission Inquiry form.
- **Branded Design**: A consistent professional color palette using Pharmacy Green (#1a5032) and Deep Red (#8c2121).
- **Campus Life**: Highlights extra-curricular talent, Pharmacy Week, and Guest Lectures.

## 🧪 Built With
- **HTML5**: Semantically structured for better accessibility.
- **CSS3**: Custom Flexbox and Grid layouts with advanced Hero blur effects.
- **Google Maps API**: Integrated map iframe for campus location.
